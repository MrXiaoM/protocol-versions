# protocol-versions
latest QQ protocol versions (release).

Update for providing basic protocol information. Check update everyday automatically.

> [!WARNING]
> 
> DO NOT USE `9.x` OR ABOVE IF YOU HAVE NOT ABILITY TO BYPASS THE CHECK.
>
